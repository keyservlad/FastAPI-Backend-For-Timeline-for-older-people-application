# AnnotatedAppAPI
# Requirements
- Python 3.6+
# Quick start for development
- Configure .env file
- pip3 install -r requirements.txt
- python3 main.py
# API documentation
- Swagger: http://SERVER_HOST:SERVER_PORT/docs
- ReDoc: http://SERVER_HOST:SERVER_PORT/redoc
# Functionality
- Database access to MongoDB
- Communicate with Front-End
